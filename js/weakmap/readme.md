- 正确使用weakmap
弱引用  2.js

1. WeakMap只接受对象作为键名

2. 所引用的对象是弱引用
弱引用：
  一个对象若只被弱引用，则被认为是不可访问的， 并因此可能在任何时刻被回收。
  2.js
  所引用的对象会被回收

  省掉手动删除对象关联数据的步骤时， 或者不想控制关联数据的生命周期时可以使用WeakMap。

  一 没有遍历操作（keys(), values(), entries()），没有size, 也不支持clear
